Install ASF (Offline)

  1. Mount the USB drive.
  2. Run the installer:
sudo bash /media/usb/asf/install-asf.sh /media/usb/asf/agent-systems-framework-YYYYMMDD.tar.gz /opt/agent-systems-framework
  1. Validate:
cd /opt/agent-systems-framework/agent-systems-framework && ./scripts/validate.sh && ./scripts/autofile init && ./scripts/health.sh
