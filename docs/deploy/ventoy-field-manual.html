Ventoy Setup

  1. Download Ventoy from https://www.ventoy.net/en/download.html
  2. Install: sudo ./Ventoy2Disk.sh -i /dev/sdX
  3. Update later: sudo ./Ventoy2Disk.sh -u /dev/sdX
  4. Copy ISOs into iso/ on the USB
Warning: Ventoy install wipes the target device. Verify the disk.

Library Blueprint

Required Core

  • Debian netinst + Debian DVD (offline packages)
  • Fedora Workstation or Server
  • Arch ISO
  • Gentoo minimal ISO + stage3 tarballs

Rescue and Repair

  • SystemRescue
  • GParted Live
  • Rescuezilla
  • Clonezilla
  • Hiren's BootCD PE (license check)
  • Memtest86+
  • Super Grub2 Disk

Privacy and Security

  • Tails ISO (verify signatures)
  • Whonix VM images (Gateway + Workstation)

Optional and Legacy

  • FreeDOS (open alternative)
  • Windows 95 ISO only if you own licensed media
  • DOSBox or similar emulator (optional)

Persistence for Live Distros

Use Ventoy persistence files so live systems can carry state between boots.

  • Create a persistence image file per ISO (size based on your needs).
  • Store persistence files under persistence/.
  • Configure mappings in ventoy/ventoy.json.
Persistence support is distro-specific. Test each live ISO.

Toolchains for the Four Agents

Carry the compilers and runtimes needed for Rust, C/C++, COBOL, and Emergent stacks.

  • Rust: rustup + stable toolchain + cargo cache
  • C/C++: gcc, clang, cmake, make, gdb
  • COBOL: gnucobol
  • Emergent: python3, pip, node, npm

See docs/deploy/toolchain-kit.md for per-distro packaging.

Package names vary by distro. Keep a manifest per family.

Web3 Auth (Brave)

Brave is the recommended browser for Web3 auth workflows. Keep offline installers on the USB.
